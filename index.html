🔗 Kết nối Google Sheets
Lưu trạng thái chăm sóc KH lên Google Sheets để cả team cùng đọc/ghi, F5 không mất dữ liệu.
1
Mở Google Sheets → Tạo sheet mới. Đặt tên tab là CareData
2
Vào Extensions → Apps Script → Xóa hết code cũ → Dán code mới bên dưới → nhấn Deploy → New deployment
⚠ Quan trọng:
 • Type: Web app
 • Execute as: Me
 • Who has access: Anyone (không phải "Anyone with Google account")
Sau đó nhấn Deploy → Authorize access → Allow
✓ Bản mới hỗ trợ sheet AssignData → chiến dịch đồng bộ cho tất cả nhân viên
3
Copy Web app URL (dạng https://script.google.com/macros/s/AKfycb.../exec) dán vào ô bên dưới rồi bấm Kiểm tra kết nối.
!
Mỗi lần sửa code Apps Script phải Deploy lại (New deployment hoặc Manage deployments → Edit). Nếu không deploy lại thì URL cũ vẫn chạy code cũ.

⚙ Quản lý Tình trạng CS
Thêm nhóm mẹ / mục con. Kéo ☰ sắp xếp. Move / chuyển đổi loại. Đồng bộ Google Sheets.
Thêm mới
Nhóm mẹ (📁) → chỉ làm tiêu đề, không chọn được. Tình trạng lá (🔹) / con (↳) → chọn được. Xóa nhóm sẽ xóa cả con.
⚙ Quản lý Trạng thái KH
Thêm nhóm mẹ / mục con. Kéo ☰ sắp xếp. Move child / chuyển đổi loại (🔄). Đồng bộ GSheets.
Thêm mới
Nhóm mẹ (📁) → chỉ làm tiêu đề, không chọn được. Trạng thái con/lá (🔹↳) → chọn được. Dùng 🔄 để chuyển lá↔nhóm, dropdown để chuyển con sang nhóm khác.
